---

The `@lando/wordpress` plugin is the official Lando integration for WordPress. It provides a `wordpress` recipe that wires up a fully configured local development environment — PHP, a web server, a database, and WP-CLI — so you can go from zero to a running WordPress site with a single `lando start`.

## Key features

- **PHP 5.3–8.5+** — pick the exact version your project requires
- **Web server** — choose between Apache (default) and Nginx
- **Database backend** — MySQL, MariaDB, or PostgreSQL
- **WP-CLI** — run `lando wp` commands directly inside the container
- **Composer** — manage PHP dependencies without a local installation
- **Xdebug** — enable step debugging with a single config flag
- **db-import / db-export** — import and export database dumps using built-in tooling

## Basic Landofile

Add a `wordpress` recipe to a `.lando.yml` file at the root of your project to get started. The defaults use PHP 8.3, Apache, and MySQL 8.0.

```yaml
name: wordpress-app
recipe: wordpress
config:
  php: '8.3'
  database: mysql:8.0
```
